# 🚀DevTrack Pro

DevTrack pro is a personal operating system for developers that manage goals, projects, tasks, and track growth over time.

## 🎯Vision
A dynamic developer profile that combines:
- Goal tracking
- Project Management
- Task Management
- Progress Insights
- Activity timeline
- Github and Leetcode,etc Integration(Future)

## 🛠️Tech Stack

### Frontend
- React
- Vite

### Backend
-Node.js
-Express
-MongoDB
-JWT
